Requires at least: 3.0
Tested up to: 5.7.1
Stable tag: 0.3

Prevent Google fonts from loading on the frontend of the website.

== Description ==

This plugin stops loading of all Google fonts, added by themes and plugins.

There are various reasons you might need this, most noticeably is performance. But also you might be developing locally, or blocking of Google's servers, etc.

== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Boom, no extra steps; enjoy!
